# mkLauncher

mkLauncher is a Qt6 based minimalist apps and files launcher for Linux and Windows that does not rely on any indexing services.

# FAQ

### Why did you write this? What's wrong with KRunner?

Because the way KRunner orders (or rather, doesn't order) the search results by relevance drives me nuts.
Also the fact that it doesn't find some stuff even though the search terms should hit it.
And the fact, that it doesn't display the path of the files it finds.
And the fact that I can't right click files in the results.
Also the fact that I have to input the search engine before the search terms.
